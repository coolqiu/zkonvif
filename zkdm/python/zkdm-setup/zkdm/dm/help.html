实现 dm 接口：
dm 根据配置，启动相应的服务，并且支持杀掉进程 ...

提供如下接口：

/dm/list: 列出该机配置的所有服务
返回：application/json

{
"result": "ok",
"info": "",
"value": {
"type":"list",
"data": [
{
"name":"event service",
"type":"event",
"enable": true,
"url": "http://172.16.1.103:10001/event"
}
{
”name":"teacher video 1",
"type":"evel",
"enable": false,
"url": "rtmp://172.16.1.103/fms...."
}
....
]
}
}


/dm/(service name)/start: 启动指定名字的服务
/dm/(service name)/stop: 停止服务
/dm/(service name)/enable: 启用服务
/dm/(service name)/disable: 禁用服务

返回：application/json
成功:
{
"result":"ok",
"info":""
}
失败：
{
"result":"error",
"info":"not found or disabled"
}
/dm//start: 启动指定名字的服务 /dm//stop: 停止服务 /dm//enable: 启用服务 /dm//disable: 禁用服务 返回：application/json 成功: { "result":"ok", "info":"" } 失败： { "result":"error", "info":"not found or disabled" } /dm/host?command=shutdown: 关机 /dm/host?command=restart: 重启计算机 返回: application/json 成功(唯一值): { "result": "ok", "info": "host is XXXing ..." } dm/host?command=type: 获取主机类型 返回: application/json 成功: { "result": "ok", "info" : "", "value": { "type":"dict", "data": { "hostType":"JP100" } } } 失败: { "result":"error", "info":"can't get host type" } /dm/host?command=reformance: 返回: application/json 成功: { "result" : "ok", "info" : "", "value" : { "type" : "dict", "data" : { "cpu_rate" : 0.02, "memory" : { "availabled" : 5344845824, "committed" : 3702054912 }, "net" : [ { "name" : "Intel[R] 82579LM Gigabit Network Connection", "sent_rate" : 0, "recv_rate" : 5150 }, { "name" : "Intel[R] Centrino[R] Advanced-N 6205", "sent_rate" : 0, "recv_rate" : 0 }, ], "disk" : [ { "name" : "_Total" "totalMB" : 719371, "freedMB" : 453204, }, { "name" : "C:" "totalMB" : 718715, "freedMB" : 452791, }, { "name" : "HarddiskVolume4" "totalMB" : 448, "freedMB" : 413, } ] } } } 失败: { "result" : "ok", "info" : "error" }
