录像服务接口:为录像提供启动、停止、暂停等操作接口，同时能提供录像参数设置接口。

一、录像操作

1）/recording/start:录像开始

2)/recording/stop:录像停止

3）/recording/pause:录像暂停

4）/recording/resume:录像恢复

返回：application/json

//成功"result"="ok","info"="",失败"result"="error","info"="失败原因"

{

"result":"ok"

"info":""

}

二、录制模式设置

//目前RecordMode支持All（全景模式）、Resource（资源模式）和Movie（电影模式）三种模式

{

"RecordMode":"All"

}

设置为电影模式的示例:/recording/setRecordMode?RecordMode=Movie

返回：application/json

//成功"result"="ok","info"="",失败"result"="error","info"="失败原因"

{

"result":"ok"

"info":""

}

三、录像文件路径

1）总路径：

{

"FileFormat":"mp4"

"ToatlFilePath":"D:/rec"

}

设置文件格式为mp4，路径存储在"D:/rec"的示例：/recording/setFileProperty?FileFormat=mp4&ToatlFilePath=D:/rec

2）子路径：

{

"SubFileFolder":"yyyy-MM-dd HH.mm.ss"

}

设置子路径为2014-08-19 08.08.08的示例：/recording/setFileFolder?SubFileFolder=2014-08-19 08.08.08

四、片头设置

1）图片：

1.图片路径：

//Key=3100用来标示片头

{

"FilePath":""

"Key":""

"StartTime":""

}

示例：/recording/OpenPicFile?FilePath=D:\image.png&Key=3100&StartTime=0

2.开启图片片头：/recording/PlayPicFile?Key=3100

//Key=3100用来标示片头

{

"Key":"3100"

}

3.关闭图片片头：/recording/StopPicFile?Key=3100

//Key=3100用来标示片头

{

"Key":"3100"

}
