录像服务接口:为录像提供启动、停止、暂停等操作接口，同时能提供录像参数设置接口。

1）调用方法：/recording/cmd

//成功"result"="ok","info"="",失败"result"="error","info"="失败原因"

{

"result":"ok"

"info":""

}

2)调用例子：

实现录像开始的操作命令：/recording/cmd?RecordCmd=StartRecord

3)命令说明：

一、 录像控制协议

1、 录像开始：RecordCmd=StartRecord

2、 录像暂停：RecordCmd=PauseRecord

3、 停止录像：RecordCmd=StopRecord

4、 恢复录像：RecordCmd=ResumeRecord

5、 片头设置：

（1） 图片：RecordCmd=OpenPicFile&FilePath={0}&Key=3100&StartTime=0

RecordCmd=PlayPicFile&Key=3100

 

RecordCmd=StopPicFile&Key=3100

（2） 视频：RecordCmd=OpenVideoFile&FilePath={0}&Key=3100&StartTime=0

                   RecordCmd=PlayVideoFile&Key=3100

RecordCmd=ChangeCard&CardPort=3100&isMain=true

 

RecordCmd=StopVideoFile&Key=3100

6、 片尾设置：

（1） 图片：RecordCmd=OpenPicFile&FilePath={0}&Key=3101&StartTime=0

RecordCmd=PlayPicFile&Key=3101

 

RecordCmd=StopPicFile&Key=3101

（2） 视频：RecordCmd=OpenVideoFile&FilePath={0}&Key=3101&StartTime=0

      RecordCmd=PlayVideoFile&Key=3101

RecordCmd=ChangeCard&CardPort=3101&isMain=true

 

RecordCmd=StopVideoFile&Key=3101

7、 录制模式：RecordCmd=SetRecordMode&RecordMode=All/Resource/Movie

8、 录像文件路径：

（1） 总路径：RecordCmd=SetFileProperty&FileFormat=mp4&TotalFilePath=D:/rec

（2） 子路径：RecordCmd=SetFileFolder&SubFileFolder=yyyy-MM-dd HH.mm.ss

9、 课程信息设置：

RecordCmd=SetCourseInfo&Department={0}&Subject={1}&CourseName={2}

&Teacher={3}&Address={4}&DateTime={5}&Description={6}

1、 是否进行某路资源的录像：

（1） 录制：RecordCmd=SetResRecordEnable&CardPort={0}&IsRecord=true

（2） 不录制：RecordCmd=SetResRecordEnable&CardPort={0}&IsRecord=false

2、 设置每一路视频的声音：RecordCmd=SetVolume&CardPort={0}&Volume={1}

3、 电影录像参数设置：RecordCmd=SetFilmParams& BitRate={0}&RecordFormat={1}

&FrameRate={2}&Width={3}&Height={4}

4、 每路资源的录像参数设置：RecordCmd=SetResParams& CardPort={0}&BitRate={1}

&RecordFormat={2}

5、 设置叠画持续时间：RecordCmd=SetDisSolve&DissolveDuraTime={0}

6、 每路源基本信息设置：RecordCmd=SetCardInfo&CardPort={0}&IsUse={1}&IsLock={2}

&CardName={3}

二、 切换控制协议

1、 切换窗口：RecordCmd =ChangeCard&CardPort=3000&isMain=true

2、 窗口添加特效：RecordCmd=SetCardEffect&CardPort={0}&IsOpen=true&EffectType={1}

         其中EffectType为0-10，

{ "FadeInOut", "0" },淡入淡出

 { "LtoR", "1" }, 由左到右

{ "RtoL", "2" }, 由右到左

{ "UtoD", "3" }, 由上到下

{ "DtoU", "4" }, 由下到上

{ "LUtoRD", "5" }, 由左上到右下

  { "RUtoLD", "6" }, 有右上到左下

{ "LDtoRU", "7" }, 由左下到右上

{ "RDtoLU", "8" }, 由右下到左上

{"WindowblindH","9"}, 水平百叶窗

{"WindowblindC","10"}, 垂直百叶窗

3、 没有特效：RecordCmd=SetCardEffect&CardPort={0}&IsOpen=false

4、 全部命令设置：

RecordCmd=SetAllEffect&EffectType={0}

三、 画中画控制协议

1、 子画面选取：RecordCmd=SetPicInPicCardPort&CardPort=3000&isMain=true

2、 模式选取：

RecordCmd=SetPicInPicMode&PicInPicMode=LeftRight/Tile&isMain=true

       如果模式为Insert模式：RecordCmd=SetPicInPicMode&PicInPicMode={0}&Pos={1},{2}&Size={3},{4}&CurrPos=LU/LD/RU/RD&isMain={5}

    如果模式为Pnp模式（全景）：

         RecordCmd=SetPnp&CardPort={0}&isMain=true（port为主画面的端口号值）

3、 更换主副画面：RecordCmd=ChangePicInPic&isMain=true

4、 设置小画面的大小和位置：RecordCmd=SetPositionAndSize&Pos={0},{1}&Size={2},{3}

&CurrPos=LU/LD/RU/RD&isMain=false

5、 关闭小画面：RecordCmd=ClosePicInPic&isMain=false

四、 获取信息类协议

1、 获取mac地址：MetaInfoCmd=GetMac

2、 获取ip地址：BroadCastCmd=GetDeviceIP

3、 获取磁盘空间状态值：RecordCmd=QueryDiskStatus

4、 获取录像时间值：RecordCmd=QueryRecordTime

5、 获取系统时间：RecordCmd=QuerySystemTime

五、 云台文件传送控制

1、 保存云台传回的文件信息

MetaInfoCmd=SaveFile&FileName=UPLOADMETA&FileContent={0}

2、 保存metainfo信息

MetaInfoCmd=SetMetaInfo&Guid={0}&LoginName={1}&CourseName={2}&Teacher={3}&Subject={4}&Grade={5}

六、 字幕台标类设置

1、 字幕：

      （1）开启：RecordCmd=SetCaption& Content={0}&Font={1}&FontColor={2}&BackColor={3}

&FontOpacity={4}&OutlineColor={5}&isMain=true

      （2）关闭：RecordCmd=CaptionClose&isMain=true

2、 台标：

      （1）开启：RecordCmd=SetLogo& PicPath={0}&PicPos={1}&PicSize={2}&PicOpacity={3}

&LogoNum={4}

      （2）关闭：RecordCmd=LogoClose&LogoNum={0}&isMain=true

七、 跟踪控制协议

1、 手动切换：RecordCmd=ManualTrace

2、 自动切换：RecordCmd= AutoTrace

3、 鼠标点击跟踪：RecordCmd=MouseTrace&Num={0}&X={1}&Y={2}

4、 变倍选择：RecordCmd=Zoom&Num={0}&ZoomNum={1}
