直播服务接口:为直播提供启动、停止、暂停等操作接口，同时能提供直播信息设置接口。

1）调用方法：/living/cmd

//成功"result"="ok","info"="",失败"result"="error","info"="失败原因"

{

"result":"ok"

"info":""

}

2)调用例子：

实现直播开始的操作命令：/living/cmd?BroadCastCmd==StartBroadCast

3)命令说明：

一、 直播控制协议

1、 直播开始：BroadCastCmd=StartBroadCast

2、 停止直播：BroadCastCmd=StopBroadCast

3、 直播信息设置：BroadCastCmd=SetBroadCastProperty&Width={0}&Height={1}&

                                        FrameRate={2}&BitRate={3}&ServerIp={4}&StreamName={5}&

                                       LivingNum={6}&LivingType={7}&Account={8}&Passward={9}&

                                       NetPort={10}&RtmpServerName={11}
