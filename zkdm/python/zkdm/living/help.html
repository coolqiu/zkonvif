直播服务接口:为直播提供启动、停止、暂停等操作接口，同时能提供直播信息设置接口。

一、获取直播列表:/living/list_streaming

返回：application/json

{

"result":"ok"

"info":""

"value":{

"type":"list"

"":[

{

"Living_id":"living0"

"Living_Name":"直播一"

"Living_Address":"rtmp://192.168.1.10/live/09700_0"

}

{

"Living_id":"living1"

"Living_Name":"直播二"

"Living_Address":"rtmp://192.168.1.10/live/09700_1"

}

}

}

一、直播操作

1）/living/(living_id)/start:直播开始

2)/living/(living_id)/stop:直播停止

返回：application/json

//成功"result"="ok","info"="",失败"result"="error","info"="失败原因"

{

"result":"ok"

"info":""

}

二、直播信息设置

//json数据格式

{

"Width":""

"Heitht":""

"FrameRate":""

"BitRate":""

"ServerIp":""

"StreamName":""

"LivingNum":""

"LivingType":""

"Account":""

"Passward":""

"NetPort":""

"RtmpServerName":""

}

例子：/living/(living_id)/setProperty?Width=1920&Height=1080&FrameRate=30&BitRate=192&ServerIP=192.168.1.10&StreamName=test&LivingNum=5&Account=admin&Passward=admin&NetPort=8888&RtmpServerName=test
