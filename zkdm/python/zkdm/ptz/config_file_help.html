config file interfaces

1 get all parameters about config file

http://ip:10003/ptz/config_file/fname/get_cfg
parameters:
  • fname: config file name
  • return type: json
  • example:http://127.0.0.1:10003/ptz/config_file/test.config/get_cfg

2 get values by keys

http://ip:10003/ptz/config_file/fname/getValuesByKeys?key1&key2&...&keyN
parameters:
  • fname:ditto
  • return type:ditto
  • return content:existing key-value pairs and not existing keys
  • example:http://127.0.0.1:10003/ptz/config_file/test.config/getValuesByKeys?hva&vva

3 set values by keys

http://ip:10003/127.0.0.1:10003/ptz/config_file/fname/getValuesByKeys?key1=value1&key2=values...&keyN=valueN
parameters:
  • fname:ditto
  • return type:ditto
  • example:http://127.0.0.1:10003/ptz/config_file/test.config/setValuesByKeys?hva=50

4 restart ptz service

http://ip:10003/127.0.0.1:10003/ptz/config_file/fname/restart
patameters:
  • fname:ditto
  • return type:ditto
  • example:http://127.0.0.1:10003/ptz/config_file/test.config/restart
