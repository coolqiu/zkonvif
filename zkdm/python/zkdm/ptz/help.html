云天控制接口:

using "http://ip:port/config" to get all configured ptz.
返回 application/json:
{
"result":"ok",
"info": "",
"value": {
"type": "list",
"data": [
{
"name": "teacher",
"config": {
"serial": "com1",
"addr": 1
}
}
{
"name": "student",
"config": {
"serial": "com3",
"addr": 1
}
}
]
}
}

using "http://ip:port/ptz/which/method?params..." to control.
如: /ptz/teacher/left?speed=1

method: left, right, up, down, stop, set_pos, get_pos, set_zoom, get_zoom
