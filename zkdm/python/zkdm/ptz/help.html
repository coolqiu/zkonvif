云天控制接口:

using "http://ip:port/config" to get all configured ptz.
返回 application/json:
{
"result":"ok",
"info": "",
"value": {
"type": "list",
"data": [
{
"name": "teacher",
"config": {
"serial": "com1",
"addr": 1
}
}
{
"name": "student",
"config": {
"serial": "com3",
"addr": 1
}
}
]
}
}

using "http://ip:port/ptz/which/method?params..." to control.
如: /ptz/teacher/left?speed=1

method: left, right, up, down, stop, set_pos, get_pos, set_zoom, get_zoom method 具体描述:

    无返回值:
  • /ptz/WHO/left?speed=N : 向左转动云台, 速度范围从 1 到 30 加快.
  • /ptz/WHO/right?speed=N: 右转
  • /ptz/WHO/up?speed=N : 向上
  • /ptz/WHO/down?speed=N : 向下
  • /ptz/WHO/stop : 停止上面四个转动
  • /ptz/WHO/set_pos?x=X&y=Y&sx=SX&sy=SY : 转动到指定位置, X,Y 为绝对位置, SX, SY 为转动速度
  • /ptz/WHO/set_zoom?z=Z : 设置倍率
    有返回值, 返回 json 格式
  • /ptz/WHO/get_pos: 返回云台当前位置, 具体格式为: { "result":"ok", "info":"", "value":{"type":"position","data":{"x":X, "y":Y}}}
  • /ptz/WHO/get_zoom: 返回倍率值, 具体格式位: {"result":"ok","info":"","value":{"type":"position","data":{"z":Z}}}
